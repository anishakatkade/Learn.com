Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using elements and tags.

Key Concepts: Tags, Elements, Attributes, Semantic HTML, and Document Structure.

HTML Tags

HTML tags are the building blocks of HTML. They define the structure and content of a web page.

Common Tags: <html>, <head>, <body>, <h1> to <h6>, <p>, <a>, <img>, <div>, and <span>.

HTML Elements

An HTML element consists of a start tag, content, and an end tag. Elements can also have attributes.

Example: <p>This is a paragraph.</p>

HTML Attributes

Attributes provide additional information about HTML elements. They are always specified in the start tag.

Common Attributes: class, id, src, href, alt, and style.

Semantic HTML

Semantic HTML uses meaningful tags to define the structure of a web page. It improves accessibility and SEO.

Examples: <header>, <footer>, <nav>, <section>, <article>, and <aside>.

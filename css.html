Introduction to CSS

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls the visual presentation of HTML elements.

Key Concepts: Selectors, Box Model, Flexbox, Grid, and Responsive Design.

CSS Selectors

CSS selectors are used to target HTML elements for styling. They can be based on element names, classes, IDs, or attributes.

Common Selectors: Element (e.g., p), Class (e.g., .class), ID (e.g., #id), and Attribute (e.g., [type="text"]).

CSS Box Model

The CSS Box Model describes the design and layout of elements. It consists of content, padding, border, and margin.

Components:

  • Content: The actual content of the element.
  • Padding: Space between the content and the border.
  • Border: A border around the padding.
  • Margin: Space outside the border.

CSS Flexbox

Flexbox is a layout model that allows you to design flexible and responsive layouts. It works with a container and its child items.

Key Properties:

  • display: flex;
  • flex-direction
  • justify-content
  • align-items

CSS Grid

CSS Grid is a powerful layout system for creating two-dimensional layouts. It uses rows and columns to structure content.

Key Properties:

  • display: grid;
  • grid-template-columns
  • grid-template-rows
  • gap
